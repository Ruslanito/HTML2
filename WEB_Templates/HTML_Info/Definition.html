Список определений.

Список определений хорошо подходит для расшифровки терминов, создания глоссария, словаря, справочника и т.д. Список определений состоит из двух элементов — термина и его определения. Сам список задается с помощью контейнера <dl>, термин — тегом <dt>, а его определение — с помощью тега <dd>. Общая структура списка определений:

<dl>
<dt> Термин 1</dt>
<dd> Определение 1</dd>
<dt> Термин 2</dt>
<dd> Определение 2</dd>
</dl>
