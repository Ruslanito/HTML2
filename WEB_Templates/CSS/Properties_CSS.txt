**** The CSS Box Model:
Content - Padding (transparent) - Border - Margin (transparent)
Top - Right - Bottom - Left (TRouBLe) свойства работают в такой последовательности.
CSS использует модель коробки для определения насколько бальшимим являются коробки и как их разместить (модель коробки ипользуется также для вывычисления высоты и ширины html-элемента).
