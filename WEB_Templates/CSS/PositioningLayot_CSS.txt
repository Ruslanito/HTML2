**** display определяет поведение (как будет отображён) отдельного элемента на странице:
- block - блочный элемент без вложений
- inline - заполняет  страницу на всю длину без прерывания
- none - скрытие элемент
- list-item
- table
- table-cell
- table-column
- grid
   CSS   #style{
      display: inline;
   }
