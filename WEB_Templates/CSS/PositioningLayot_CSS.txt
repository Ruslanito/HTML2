**** display определяет поведение (как будет отображён) отдельного элемента на странице:
- block - блочный элемент без вложений
- inline - заполняет  страницу на всю длину без прерывания
- none - скрытие элемент
- list-item
- table
- table-cell
- table-column
- grid
   CSS   #style{
      display: inline;
   }
****  visibility - определяет видимость или невидимость элемента:
- visible - элемент виден
- hidden - элемент не виден, но отображается как пустой блок на странице
**** positioning elements - позволяет размещать элементы и определяет что будет при наложении или превышении размера:
- top  - размещение относительно верха
- right - размещение относительно правой части
- bottom - размещение относительно низа
- left - размещение относительно левой части
   - static - выводится в потоке документа как обычно (свойства размещения игнорируются)
   - fixed - выводится с фиксированной позицией относительно окна браузера и не будет двигаться даже при прокрутке окна броузера (свойства размещения применяются). Элементы с фиксированной позицией убираются из нормального потока. Документ и другие элементы ведут себя так, будто элемент с фиксированной позицией не существует. Элементы с фиксированной позицией могут перекрывать другие элементы.
   - relative - выводится относительно исходного места (свойства размещения применяются). Содержимое относительно позиционированных элементов может перемещаться и перекрывать другие элементы, но зарезервированное место для элемента сохраняется в нормальном потоке.
   - absolut - элемент абсолютного позиционирования расположен относительно первого родительского элемента, который имеет любое, но не нормальное, положение. Если таковой элемент не найден, то содержащий блок это <html>.
   Абсолютно позиционированные элементы убраны из нормального потока. Документ и другие элементы ведут себя так, будто абсолютно позиционированные элементы не существуют.
   CSS   p.position_static {
            position:static;
            top: 30px;
            right: 5px;
            color: red;
         }
         p.position_fixed {
            position: fixed;
            top: 30px;
            right: 5px;
            color: red;
         }
         span {
            background: green;
            color:white;
            position: relative;
            top: 150px;
            left: 50px;
         }
**** float элемент может быть выровнен по краям, позволяя всем остальным элементам обтекать его вокруг(изображения, разметка...):
- left - выравнивают элементы по левому краю
- right - выравнивают элементы по правому краю
- none - (по умолчанию) гарантирует, что элемент не будет обтекать
   HTML  <p><img src="css_logo.png" />
         This paragraph has an image that is floated to the <strong>right.</strong> It is highly recommended to add a margin to images so that the text does not get too close to the image. If you want your text to be easily read, you should always add a few pixels between words and borders, images, and other content.
         </p>
   CSS   img {
            float: right;
         }
   Элементы обтекаются горизонтально, это значит, что элемент может быть выравнен по левому, либо по правому краю, не по верху или низу.
   Элементы Рядом Друг с Другом.
   Если вы размещаете несколько обтекаемых элементов один за другим, они будут обтекаться один за другим, если достаточно места. Например, в разметке печати, изображения могут быть установлены на страницу таким образом, чтобы текст обтекал их по мере необходимости.
   CSS   img {
            float: left;
            width: 120px;
            margin-right: 10px;
         }
         p {
            width: 120px;
            float: left;
         }
**** clear отмена действия свойства Float - элементы, которые следуют за обтекаемым элементом будут протекать вокруг него. Чтобы избежать этого, используйте свойство clear. Свойство clear устанавливает, с какой стороны элемента запрещено его обтекание другими элементами.
- left - нет обтекания элемента по левому краю
- right - нет обтекания элемента по правому краю
- both - нет обтекания элемента по правому и левому краю
- none - обтекание будет с двух сторон
 В нижеуказанном примере, если мы установим свойство float на div, то только параграф, который следует после div будет обтекать изображение.
   HTML  This paragraph is above the div element
         and is not affected by the float right property.
         <br /><br />
         <div class="floating">
            <img src="css_logo.png" />
         </div>
         This paragraph comes after the div element
         and is affected by the float right property.
         <br/><br class="clearing"/>
         This paragraph is out of the floating group
         and is not affected by the float right property.
         <br/> <br/>
   CSS   .floating {
            float: right;
         }
         .clearing {
            clear: both;
         }
***** overflow - каждый элемент на странице является блоком. Если высота блока не установлена, то она будет расти, пока не будет способна вместить необходимый контент. overflow определяет поведение, которое возникает, когда содержимое элемента выходит за область элемента блока^
- visible - видимо всё даже за границей блока
- scroll -добавление полосы проктутки (горизонтальную, вертикальную)
- hiden - отображается только область внутри блока, отсаьное будет скрыто
- auto - полоса прокрутки по необходимости (по-умолчанию)
   CSS   div {
   width: 150px;
   height: 150px;
   background-color: LightBlue;
   float: left;
   overflow: hidden;
         }
**** z-index - Когда элементы расположены вне нормального потока, они могут накладываться на другие элементы, оно определяет порядок накладывания элементов (какой из элементов будет расположен впереди, или сзади по отношению к другим).Присваивание высшего z-index значения к синему div и меньшего z-index значения к красному div приведет к выделению на передний фон этого элемента:
   CSS   .blue {
            z-index: 3;
            position: relative;
         }
         .red {
            z-index: 2;
            position: relative;
         }
Свойство z-index работает только с позиционированными элементами (position:absolute, position:relative, или position:fixed).
